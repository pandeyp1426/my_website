Projects

Chuddrey Web Application

The Chuddrey Restaurant web application is an interactive online food ordering platform designed to showcase my skills in full-stack web development. Built with a combination of HTML, CSS, JavaScript, PHP, and MySQL, this project delivers a seamless user experience, from browsing menu items to placing delivery orders as a part of an academic project.

Project Features

  • User Authentication: Users can securely sign in or create an account using a registration form. The login and registration system is built with PHP and MySQL, ensuring secure data handling and authentication.
  • Dynamic Menu Display: The application displays a visually engaging food menu, complete with images, prices, and descriptions for each item. Users can select items to add to their cart, making the ordering process intuitive and interactive.
  • Shopping Cart and Order Summary: Selected menu items are added to a cart, where users can review and modify their order. JavaScript enhances interactivity by dynamically updating the cart and providing immediate feedback on user actions.
  • Order and Delivery Information: A detailed form allows users to input delivery information and choose their preferred payment method (MasterCard, Visa, or American Express). All information is processed and stored securely in a MySQL database using PHP and SQL for data management.
  • Database-Driven Backend: The application utilizes a MySQL database with multiple linked tables to store user accounts, menu items, and order details. PHP and SQL queries manage user authentication, data retrieval, and order storage, creating a robust and secure backend.

Technologies Used

  • Frontend: HTML, CSS, JavaScript for UI design and interactive elements.
  • Backend: PHP for server-side processing and SQL for managing database interactions.
  • Database: MySQL for persistent storage of user, menu, and order data.
  • Hosting: Connected to an online MySQL database, using a free hosting service for real-time data management.
